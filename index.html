Save the Mortys

Save the Morty's is Quiz/Canvas game. Two players will sign up to play a quiz game similiar to jepardy. The winner will then get to play the Canvas game to save the Morty's. Save the Morty's uses Html, Css, JS, Jquery, and Canvas.

Technologies:

Html, Css, Javascript, jQuery, Node, Express, Nodemon, JSON and Open Trivia Database Api

Pet Search

Pet Search is a web app where users can sign up and then search for adoptable pets in there area; where they can save the pet's info. The site has full CRUD routes in respect to the user and pets models. The site uses Node, Express, Postgres, JS, Jquery, Petfinder api and Materialize.

Technologies:

Javascript, Node, Express, Bcrypt, Geocoder, Sequelize, SQL/Postgres, and Petfinder Api

Ryde

Tired of battling traffic or riding the bus? Ryde brings carpooling into the 21st century, making it easy to get to work, weekend destinations, and more. And if you know that you'll be driving, why not pick up some fellow travellers on the way to share the costs?
Ryde is a group project built in collaboration with Sean Cesmat, Scott Rosehart, and Brant Porter.

Technologies:

Materialize, Node, Express, Bcrypt, Axios, Mongo/Mongoose, React/Redux

DoshTracker

An web app for tracking income. When individuals are in an industry where they are paid in cash ('dosh') like bartenders, it can be hard to know how much to need to save. This app helps to tracker your money and figure out how much you should save. I used Pyhton and Django on the backend and React for the frontend.

Technologies:

Python, Django, Bower, D3, NVD3, jQuery, Underscore
